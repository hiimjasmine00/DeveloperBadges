# Developer Badges
Badges for every* Geode mod developer.\
\* Every one I could find, at least.

## Features
- Clickable badges displayed on a user's profile and comments
- Toggleable comment coloring based on a user's badges:
  - Mod Developer
  - Verified Developer
  - Index Staff
  - Lead Developer
